# Utility Hub

A full-stack web application built with **Flask** (backend) and **Flutter** (frontend) that brings together a collection of practical tools I regularly use and find helpful in day-to-day work.

## Features

### 📄 PDF to Word Converter
Convert PDF documents into editable Word files quickly and efficiently.

### 🔢 Word Counter & Text Analyzer
Analyze text with useful statistics, including:
- Word count
- Character count (excluding spaces)
- Sentence count
- Paragraph count

### 📊 JSON to Excel Converter
Upload JSON data and export it into a structured Excel spreadsheet for easier analysis and reporting.

## Tech Stack

### Frontend
- Flutter
- Responsive web interface
- REST API integration

### Backend
- Flask
- File processing and conversion services
- API endpoints for tool functionality
